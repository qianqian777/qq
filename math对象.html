Math——数学函数和常量

概要：

  • Math是一个对象，其属性为若干有用的函数和常量。他和Date、String不同，Math不是对象的类。没有Math()构造函数，只是简单的函数，而不是对某个对象进行操作的方法。具体如下：
  • 01、Math.constant; Math对象常量;
  • 02、Math.function(); Math对象函数;

常量：

  • 01、Math.E：常量e，自然对数的底数；
  • 02、Math.ln10：10的自然对数；
  • 03、Math.ln2：2的自然对数；
  • 04、Math.log10E：e以10为底的对数；
  • 05、Math.log2E：e以2为底的对数；
  • 06、Math.PI：常量π；
  • 07、Math.sqrt1/2：2的平方根的倒数；
  • 08、Math.sqrt2：2的平方根；

静态函数：

  • 01、Math.abs()：计算绝对值；
  • 02、Math.acos()：计算反余弦值；
  • 03、Math.asin()：计算正弦值；
  • 04、Math.atan()：计算反正切值；
  • 05、Math.atan2()：计算X轴到指定点的角度；
  • 06、Math.sin：计算正弦值；
  • 07、Math.cos：计算余弦值；
  • 08、Math.tan()：计算正切值；
  • 09、Math.exp()：计算e的乘方；
  • 10、Math.log()：计算自然对数；
  • 11、Math.max()：返回两个数中较大的那个；
  • 12、Math.min()：返回两个数中较小的那个；
  • 13、Math.pow()：计算x的y次方；
  • 14、Math.sqrt：计算平方根；
  • 15、Math.round()：四舍五入；
  • 16、Math.floor()：对一个数字向下取整；
  • 17、Math.ceil()：对一个数字向上取整；
  • 18、Math.random()：计算一个随机数；

  6.       for (var i = 0 ; i <= arr.length; i++) {
  7.            if(Nmin > arr[i]) {
  8.                  Nmin = arr[i];

  10.            }
  11.            if(Nmax < arr[i]){
  12.                  Nmax = arr[i];

  14.            }
  15.       };
  16.       //方法二：Math对象
  17.       var max = Math.max.apply(null, arr);
